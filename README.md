# 🍎 macOS-Inspired Developer Portfolio

An interactive portfolio website designed to feel like a real macOS desktop experience.

Instead of a traditional website, this portfolio mimics a desktop environment with draggable folders, windows, widgets, and even a mini game.

---

## ✨ Features

### 🖥️ macOS UI Experience
- Desktop-style layout with folders
- Draggable windows (like real macOS)
- Dock navigation with app-style icons
- Smooth animations and transitions

### 📁 Interactive Desktop
- Click and drag folders anywhere
- Double-click to open sections
- Realistic OS-like interactions

### 🪟 Window System
- Draggable windows
- Multiple windows open at once
- Active window focus (z-index handling)

### 🧊 Widgets
- ⏰ Live clock widget
- 💬 Random developer quote widget (click to refresh)

### 🎮 Mini Game
- “Catch the Bug” arcade game
- Click bugs to increase score
- Fun interactive element inside the portfolio

### 📂 Sections
- Projects (Finder-style)
- About Me (Safari-style)
- Skills (Terminal-style)
- Experience (Notes-style)
- Resume (Preview-style)
- Contact (Mail-style)

---

## 🚀 Tech Stack

- **Frontend:** React / JavaScript
- **Styling:** Tailwind CSS / CSS
- **Animations:** Framer Motion (optional)
- **Game:** JavaScript (or Kaboom.js)

## 🎯 Inspiration

- macOS UI/UX design

- Interactive portfolios

## 🧠 Why This Project?

Most portfolios are static and predictable.

This project aims to:
- Stand out visually
- Show creativity + technical skill
- Provide an interactive user experience
